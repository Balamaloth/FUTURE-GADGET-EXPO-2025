 🌐 Future Gadget Expo 2025
Where Innovation Meets Tomorrow

The Future Gadget Expo 2025 is a modern and responsive event website designed to showcase futuristic technologies, visionary speakers, and groundbreaking innovations in AI, space technology, and biotechnology.

🚀 Project Overview

This website provides an engaging digital experience for the Future Gadget Expo 2025, featuring:

Interactive event highlights

Speaker profiles and sessions

Schedule previews for all three event days

Registration and ticketing information

Responsive design for all devices

🧠 Key Features

🌟 Modern UI Design: Clean and elegant layout with smooth gradients.

🧬 Event Highlights: AI, Space, and Biotech sections with action buttons.

🎤 Featured Speakers: Dynamic section showcasing key industry leaders.

📅 Event Schedule: Organized day-wise event preview.

💻 Fully Responsive: Works seamlessly on desktop, tablet, and mobile.

🛠️ Tech Stack

HTML5

CSS3 / TailwindCSS
Responsive Web Design Principles
